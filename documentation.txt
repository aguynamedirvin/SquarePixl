Make to have SASS, Bower and Grunt (and Grunt-CLI) installed to download project required files and folders. Once you have the
required tools, from a the Terminal run "npm install" and "bower install" on the project directory.


You can grab all the tools from: https://www.npmjs.com
Node-SASS: https://github.com/sass/node-sass


Commands:

	- To watch SASS / JS files and compile at the same time
		- 'grunt'

	- For Production
		- 'grunt production'

	- To compile SASS files
		- 'grunt build'

	- To compile JS files
		- 'grunt uglify'


Gruntfiles.js
	- All tasks that have the 'build' argument are the settings that will be used for production use



You can configure the themes break points, colors, fonts and more in the '_config.scss' file located inside the 'src/stylesheets' folder.

Structure:
	
	- SRC:

		- js

			- vendor: Third-party JS

		
		- stylesheets:

			- _config.scss: The configuration file

			- vendor: Third-party CSS.
